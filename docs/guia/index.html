⌨️ Cómo arrancar desde el USB

ℹ️ ¿Qué es la BIOS y qué es el menú Boot?
Son dos menús distintos que aparecen al encender, ANTES de que cargue Windows:
• Menú de arranque (Boot): una lista rápida para elegir "arranco desde el USB esta vez". Es lo que normalmente quieres. Suele ser F8, F11, F12 o Esc.
• BIOS/UEFI: los ajustes internos del PC (orden de arranque, Secure Boot...). Suele ser Supr (Del) o F2.
👉 Para instalar Windows normalmente basta con el menú Boot y elegir el USB. Solo entras a la BIOS si necesitas desactivar Secure Boot o cambiar ajustes.
ℹ️ No consigo entrar / el teclado no responde
A veces el teclado no responde tan pronto, o el PC arranca demasiado rápido. Prueba esto desde el Windows que tienes ahora:
  1. Con Mayús (Shift): mantén pulsado Shift y haz clic en Inicio → Apagar → Reiniciar → menú azul → Solución de problemas → Opciones avanzadas → Configuración de firmware UEFI.
  2. Con un comando: abre CMD y escribe shutdown /r /o /t 0.
  3. Desde Ajustes: Configuración → Sistema → Recuperación → Inicio avanzado → Reiniciar ahora.
Desde ese menú azul entras directo a la BIOS sin pelearte con las teclas. 😉
📋 Cómo ordenar el arranque (que el USB sea el primero)
Una vez dentro de la BIOS:
  1. Busca la sección "Boot" (Arranque), también llamada Boot Order o Boot Priority.
  2. Pon tu USB como primera opción (Boot Option #1). Se mueve con las teclas + / −, F5 / F6, o arrastrando.
  3. Guarda y sal (normalmente F10). El PC arrancará desde el USB y empezará la instalación.
💡 ¿No sabes cuál es tu USB? Suele salir con su marca (Kingston, SanDisk...) o como "UEFI: nombre-del-USB". Si aparece una opción que pone UEFI, elige esa.
🔒 Secure Boot (si sale error "Secure Boot Violation")
¿Qué es? Una protección de la BIOS que solo deja arrancar sistemas "firmados" por Microsoft. A veces impide arrancar desde el USB, por eso hay que apagarlo un momento.
  1. Entra al BIOS (tecla de tu marca, tabla de arriba).
  2. Busca la opción "Secure Boot" (pestaña "Boot" o "Security").
  3. Cámbiala a "Disabled".
  4. Guarda y sal (normalmente F10).
  5. Intenta arrancar desde el USB de nuevo.
✅ ¿Hay que volver a activarlo después? Sí, es recomendable: con Windows ya instalado, vuelve a la BIOS y pon Secure Boot otra vez en "Enabled". Da algo más de seguridad y algunos juegos/anticheats lo piden.
